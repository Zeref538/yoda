Recipes
A recipe is a cleaning plan you already approved — save it once, replay it on next week's file, or feed it to yoda watch for a self-cleaning drop folder.
how to use a recipe
  1. Clean a file the way you like — approve the AI's steps (and any manual edits count too).
  2. Click Save applied steps as recipe above → a yoda_recipe.json downloads. It's just the ordered list of tools + params, no data inside.
  3. Next time, either drop a new file here and click Apply a recipe file…, or run it from a terminal:
    yoda clean next_week.csv --recipe yoda_recipe.json
  4. To auto-clean every file dropped in a folder:
    yoda watch ./inbox --recipe yoda_recipe.json
    Files that don't match the recipe are quarantined, not guessed at.
Recipes are how you turn a one-off cleanup into a repeatable pipeline — same steps, same order, zero AI needed on replay.
